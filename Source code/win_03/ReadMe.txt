Môi trường: Visual Studio 2013

Các chức năng thực hiện được:
	1. Tạo ra giao diện có một edit control để nhập liệu. Có lệnh New để tạo 1 tập tin mới.
	2. Lưu lại dưới dạng file text.
	3. Mở một tập tin, đọc nội dung và hiển thị để chỉnh sửa tiếp.
	4. Hỗ trợ các thao tác Cut - Copy - Paste - Delete - SelectAll.Tạo hộp Font để chỉnh sửa font, size của đoạn text.
	
	7.Các luồng sự kiện phụ:
		-  Người dùng bấm close hoặc exit sẽ hỏi xem có save file hay không
		- Có các phím tắt như New: ctrl+ N, Open: Ctrl+O, Save: Ctrl+S, Select All: Ctrl+A
		- Đọc và ghi file unicode
